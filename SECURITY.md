# Security Policy

Probity is a protocol specification and reference verifier.

Security issues may include:
- Canonicalization inconsistencies
- Signature verification flaws
- Hashing weaknesses
- Verification result ambiguity
- Vector inconsistencies

Please report security issues privately to:
dev@onkura.ai

Do not open public issues for undisclosed vulnerabilities.

The reference verifier is not production infrastructure.
It is a compliance tool.
